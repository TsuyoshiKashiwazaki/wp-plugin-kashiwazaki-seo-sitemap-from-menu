WordPressのナビゲーションメニューからHTML形式のサイトマップを生成するプラグインです。

== Description ==

Kashiwazaki SEO Sitemap from Menuは、WordPressのナビゲーションメニューを利用して、SEOに配慮したHTMLサイトマップを簡単に作成・表示するためのプラグインです。

ショートコード `[menu_sitemap]` を固定ページや投稿に配置するだけで、サイトのメニュー構造を階層的に表示できます。

= 主な機能 =

*   ショートコードで簡単に設置可能
*   複数のリストスタイル（黒丸、白丸、四角、なし、ディレクトリツリー）
*   Yoast SEO、All in One SEO、Rank Math、SEOPressなどのSEOプラグインと連携し、Descriptionをツールチップで表示
*   カスタムCSSによるデザイン調整
*   多言語対応（日本語翻訳ファイルを同梱）

== Installation ==

1. プラグインのZIPファイルをダウンロードします。
2. WordPress管理画面の「プラグイン」>「新規追加」>「プラグインのアップロード」からZIPファイルをアップロードします。
3. プラグインを有効化します。
4. 「設定」>「Kashiwazaki SEO Sitemap from Menu」から設定を行います。
5. 固定ページなどに `[menu_sitemap]` ショートコードを配置します。

== Frequently Asked Questions ==

= どのようにサイトマップを表示しますか？ =

固定ページや投稿のエディタにショートコード `[menu_sitemap]` を入力してください。

= 特定のメニューを表示するには？ =

ショートコードに `menu` パラメータを追加します。例: `[menu_sitemap menu="main-menu"]`

= デザインを変更できますか？ =

設定画面の「カスタムCSS」フィールドにCSSを入力することで、デザインを自由にカスタマイズできます。
