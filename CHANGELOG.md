# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-26

### Added
- Initial release of the plugin.
- Shortcode `[menu_sitemap]` to display sitemap from menu.
- Settings page for default menu, list style, indentation, and custom CSS.
- Tooltip feature for page descriptions (supports Yoast SEO, AIOSEO, Rank Math, SEOPress).
- Multiple list styles: Disc, Circle, Square, None, and Directory Tree.
